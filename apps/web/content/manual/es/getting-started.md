# Primeros pasos

Bienvenido al Manual de lazyit — la documentación propia del producto, publicada junto con el
código y servida desde una ruta **pública, sin inicio de sesión** (ADR-0062). Es distinto de la
Base de Conocimiento: el Manual documenta *lazyit en sí*, la BC documenta *tu parque*.

Esta página existe solo para validar el andamiaje. Ten en cuenta que las páginas del Manual son
markdown de producto sencillo: los tokens exclusivos de la BC como los enlaces wiki
`[[algun-slug]]` y los chips de secreto `{{ lazyit_secret.HANDLE }}` se muestran aquí como
**texto literal**, nunca como elementos activos — el Manual está libre de secretos por diseño.
